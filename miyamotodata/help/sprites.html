Sprites


Sprite Paletten

Die Sprite-Palette ermöglicht es dir, Sprites zum Malen auszuwählen. Sobald ein Sprite ausgewählt ist, erscheinen einige Datenfelder am unteren Rand der Palette. Dies ermöglicht es dir, die Anfangsdaten für ein Sprite festzulegen. Male ein Sprite, das in der Palette ausgewählt wurde, mit einem Rechtsklick. Sprites können nicht in der Größe verändert werden, so dass das Ziehen nichts bewirkt. Sprites werden mit den unten ausgewählten Eigenschaften gemalt. Du kannst beliebig viele Sprites in einem Level haben - zu viele auf dem Bildschirm können zu einer Verlangsamung des Spiels führen.

Du kannst auch die Eigenschaften eines Sprites jederzeit ändern, indem du ein Sprite auswählst. Dies führt dazu, dass das Sprite-Eigenschaftenfenster geöffnet wird, so dass du die Daten nach Belieben ändern kannst. Die sinnvolle Verwendung von Sprite-Eigenschaften ist der Schlüssel, um Spaß und Freude an Levels zu haben.


Sprite Properties Box

This box will pop up whenever you select a sprite. Feel free to resize, move, or dock this window. It will use a variety of combo boxes, check boxes, and spin boxes to get user input from you as to how exactly you want a sprite to be.

There is also a raw data modifier, which will allow advanced users to attempt to determine if there are any unknown options available. All sprite data is kept within the spritedata.xml file inside the reggiedata folder. These same mysterious advanced users can edit it to add new features to the sprite property box. Advanced users can also create stars from lead, and pull fireflowers out of their sleeves.


Sprite Standorte

In einigen Levels siehst du violette Rechtecke mit einer Zahl an der Ecke, die du ziehen und in der Größe verändern kannst - das sind Orte für bestimmte Sprites. Einige Sprites (wie Porcu-Puffer) erlauben es dir, einen Speicherort zu erstellen und damit anzugeben, in welchem Bereich das Sprite arbeiten soll. Andere (wie Lava, Wasser, Treibsand und Wolken) erscheinen innerhalb des Bereichs, der durch den Standort belegt ist.

Um sie zu bearbeiten, kannst du einfach einen auswählen und verschieben oder die Größe mit dem Grabber unten rechts ändern. Eine Zahl wird oben links an jedem Ort angezeigt; diese Zahl kann in den Eigenschaften für zugehörige Sprites verwendet werden.

Um einen neuen Ort zu erstellen, klick auf den Sprite Picker und wähle die Option "Malt neue Sprite-Bereiche/Location ein.". Du kannst dann mit der rechten Maustaste auf die Leinwand klicken und ziehen, um einen zu malen - eine ID wird ihr automatisch zugewiesen.
